What is .me?
.me is a semantic knowledge graph engine that lets you build AI systems where everything is understandable, reactive, and yours.

The kernel figures out what depends on what, and when anything changes, every dependent value updates automatically. No polling. No event listeners. No manual wiring. A single sensor mutation in a city of 1 million nodes propagates to exactly the 6 nodes that care about it — in 0.256ms. The same object means different things to different robots depending on their context. Private data stays structurally invisible — not hidden behind a permission check, just honestly absent. And every derived value comes with a full explanation of how it got there.
Data that thinks. Logic that explains itself.
